Type

適用されるディストーションアルゴリズムのタイプ。以下のオプションから選択する。

  • None:None（なし）： ディストーションは適用されない。
  • Overdrive:Overdrive（オーバードライブ）： スムースなブルースタイプのディストーション。
  • Heavy:Heavy（ヘビー）：高周波数の高調波を導入するハードディストーション。
  • Clip:Clip（クリップ）： トランジションのないハードクリッピング。しばしば不快なスタティックノイズを引き起こす（クリッピングディストーション）。
  • Fuzz:Fuzz（ファズ）： 正と負の値に対して異なる閾値を持つ非対称のハードおよびソフトクリッピング。このタイプのディストーションは、高調波を多く導入するので、複雑なトーン入力では、急速に不協和音が聞こえる場合がある。

Default value: None
