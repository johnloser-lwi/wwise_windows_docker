Compressor Link Mode

Compressor 链接模式。指定要采用哪种模式向各个声道应用压缩。若使用 Mastering Suite 处理基于对象的音频，则始终将 Link Mode 设为 All channels，同时不沿用任何用户定义的设置。

  • No link（不链接）：单独向每个声道应用压缩。
  • All channels（所有声道）：统一向所有声道应用最大压缩。
  • Partial link（部分链接）：使用 Link Strength 值来控制声道的压缩。对于对应声道，按照最大值应用压缩；对于其他声道，则按照相对于最大值的 Link Strength 比率应用压缩。

Default value: All channels
