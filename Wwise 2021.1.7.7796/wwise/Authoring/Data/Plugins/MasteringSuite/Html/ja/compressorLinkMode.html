Compressor Link Mode

チャンネル同士に対し、コンプレッションの適用モードの指定。Mastering Suiteでオブジェクトベースオーディオを使う場合は、Link Modeは常に ‘All channels’ の設定となり、ユーザーが選択した設定は全てオーバーライドされる。

  • No link: 各チャンネルに、個別にコンプレッションを適用する
  • All channels: 最高のコンプレッションを、全チャンネルに、等しく適用する
  • Partial link: Link Strength値に従ってチャンネルのコンプレッションを制御する方法で、最高のコンプレッションを該当するチャンネルに適用し、それにLink Strengthレシオを適用したものを、残りのチャンネルに適用する

Default value: All channels
