デスティネーションの再生を開始する、ミュージックオブジェクト内のポイントを設定する。以下のオプションから選択する。

  • Start of Playlist:プレイリストの最初の項目にジャンプする。
  • Specific Playlist Item:プレイリストの指定した項目にジャンプする。プレイリストのどの項目にジャンプするかは、ブラウズボタンをクリックして指定する。本オプションは、デスティネーションオブジェクトがMusic Playlist Containerの場合に限り選択できる。
  • Last Played Segment:最後に再生したセグメントにジャンプする。
  • Next Segment:プレイリストの最後に再生したセグメントの次にくる項目にジャンプする。

Default value: Start of Playlist
