Scope

  • Global: 게임 내에서 쓰이는 컨테이너의 모든 인스턴스가 다른 게임 오브젝트에서도 반복되는 것을 막기 위해 하나의 오브젝트로 취급하는 것을 의미합니다. 예를 들어서, 게임 내에서 한 캐릭터 집단이 모두 같은 문장들을 사용한다고 할 때, Global 옵션을 통해 Random, Shuffle, 및 Step 기능을 사용하여 캐릭터가 말할 때 마다 반드시 다른 문장을 말하도록 할 수 있습니다.
  • Game object: 각 컨테이너의 인스턴스는 개별의 독립체로 간주되기 때문에 게임 오브젝트간의 공유가 일어나지 않습니다.

Default value: Global
