Limiter Mode

  • Hard-Knee Limiter: オーディオシグナルを、スレッショルド値でカットオフする。
  • Soft-Knee Limiter: スレッショルドを超えることを一時的に許可し、オーディオシグナルを上下させ、指定されたアタックまたはリリースの時間だけかけて、徐々にスレッショルドレベルまで変化させる。
  • Advanced Limiter: 2.67ミリ秒のレイテンシを導入し、インプットがスレッショルドに到達する前に、アタックを自動的に調整し、ピークの発生にもスムーズに対応する。シグナルは、スレッショルド未満で維持される。

Default value: Soft
