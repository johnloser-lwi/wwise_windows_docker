Compressor Link Mode

채널과 관련하여 컴프레션을 적용하는 데에 사용되는 모드를 지정합니다. Mastering Suite에 오브젝트 기반 오디오를 사용할 경우, Link Mode가 항상 ‘All channels’로 설정되며 사용자가 정의한 설정을 항상 덮어쓰게 됩니다.

  • No link: 컴프레션이 각 채널에 개별적으로 적용됨
  • All channels: 최대 컴프레션이 모든 채널에 동등하게 적용됨
  • Partial link: Link Strength의 값을 사용하여 채널의 컴프레션을 제어함. 상응하는 채널에 최대 컴프레션을 적용하고 최대값에 대한 Link Strength 비율을 다른 채널에 적용함

Default value: All channels
