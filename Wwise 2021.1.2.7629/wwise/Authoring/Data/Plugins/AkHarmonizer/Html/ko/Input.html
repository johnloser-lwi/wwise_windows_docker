Input

이 매개 변수는 어떤 채널이 하모나이저에 의해 처리될 것인가를 정합니다.

Values:

  • As Input: 모든 입력 채널을 처리합니다 (기본 설정).
  • Mono-Center: 중앙 채널만 처리합니다. 모노 신호 또한 처리합니다.
  • Stereo: 좌측과 우측 스피커만 처리합니다.
  • L-R-C: 중앙 스피커가 있을 경우 중앙 스피커까지 모든 앞쪽 스피커를 처리합니다.
  • L-R-Ls-Rs: 앞쪽과 뒷쪽 스피커 쌍을 처리합니다.
  • L-R-C-Ls-Rs: 모든 전채 대역 채널을 처리합니다.
  • Left Only: 왼쪽 채널만 하모나이저에 의해 처리됩니다. 이 모드는 Effect가 버스에 삽입되어 이 Effect에 라우팅된 오디오 컨텐츠가 공간이동 또는 패닝을 사용하지 않는 모노음일 경우에 CPU 사용을 절약하려고 할 때에 유용합니다. 화음화된 좌측 채널은 좌측과 우측 앞면 채널의 변조 경로로 모두 적용됩니다. 반면에 원본 경로는 원본 채널 컨텐츠를 그대로 유지합니다.

Default value: As Input
