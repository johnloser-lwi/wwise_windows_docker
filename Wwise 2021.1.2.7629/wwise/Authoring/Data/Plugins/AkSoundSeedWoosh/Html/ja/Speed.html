Object Speed

最速走行するディフレクタが空中を移動する時の平均速度。最速走行のディフレクタが存在する場合、これは、アンカーポイントから一番遠いものになる。

オブジェクトの速度は、シーン内のディフレクタに適用されるエネルギーのオフセットによって解釈される。このプロパティは、基本的にはディフレクタの共振周波数にスピードが与える影響を定義。

デフォルトでは、 Automate オプションが Object Speed プロパティに選択される。オートメーションカーブには、 (t=0, 0)、(t=0.25, 100)、(t=0.5, 0) の3つのポイントが含まれなっている。

Default value: 0
Range: -50.0 to 50.0
