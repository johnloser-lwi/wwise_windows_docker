Pre-delay

信号が残響ユニットに入る前に発生する遅延。最初のエコーが聞こえるまでにより時間のかかる大きな部屋をシミュレートするためにプリディレイタイムを使用することができる。最初の反射音は、プリディレイタイムとリバーブ用に指定された最短のディレイラインの合計に基づいた時間間隔で発生する。Pre DelayはExclusive RTPCに対応しているが、再生前の初期化だけに使用すること。再生中に、RTPCを使いPre Delayをリアルタイムで変更すると、ディレイバッファが際限なく再初期化されてしまい、急な沈黙やリバーブ信号の変化が生じてしまう。

単位: s

Default value: 0
Range: 0 to 1
