Filters Enable

オーディオ信号に次のフィルタを適用する:

High-Pass Filter (HPF) - HPF 制御は、オーディオ信号に適用されるハイパスフィルタの特性を定義。ハイパスフィルタは、低周波数性のランブルや不要なボーカル破裂音、DCオフセットを除去するのに有用。‘futzing’ アプリケーションでは、ローエンドの周波数成分の損失は電話やトランシーバー、ラジオ、メガホンなどの多くの通信デバイスに典型的なもの。

Low-Pass Filter (LPF) - LPF制御は、オーディオ信号に適用されるローパスフィルタの特性を定義。ローパスフィルタはカセットテープのヒスノイズ、滝の音、一般的な背景雑音などのバックグラウンドノイズに有用。

Default value: true
