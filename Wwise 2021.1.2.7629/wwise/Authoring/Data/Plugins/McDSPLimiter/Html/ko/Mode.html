Mode

다양한 입력 신호와 애플리케이션에 적합하기 위한 다양한 제한 스타일을 생성하기 위해 쓰이는 2차적 감지의 유형을 말합니다. 다음은 사용 가능한 모드 옵션입니다:

  • Clean - 가장 투명한 제한 모드입니다. 신호 레벨은 측정 가능한 디스토션의 최소량으로 조절됩니다.
  • Soft - Clean 모드보다 약간 더 소리가 크지만 또한 아주 투명합니다.
  • Smart - 신호 레벨을 Soft 모드보다 더 증가시키며 신호 디스토션을 최소화하는 지능형 제한.
  • Dynamic - Smart 모드보다 소리가 더 크며 특정 상태에서 약간의 압축을 더합니다.
  • Loud - 신호 디스토션을 최소로 줄이면서 가능한 소리를 최대로 크게 높입니다.
  • Crush - Loud 모드 보다는 소리가 더 크며 신호에 약간의 디스토션이 생깁니다.

Default value: Clean
