Marker Input Mode

마커의 관리 방법을 결정합니다. 다음의 옵션 중에서 선택하실 수 있습니다:

  • Import From File을 선택하면 원본 오디오 파일에서 저장된 마커가 사용됩니다.
  • Detect From Transients을 선택하면 오디오 파일에서 과도 신호(transient)의 시작 지점을 자동으로 감지하여 감지된 지점에 마커를 추가합니다. 이 옵션을 선택하면 감지 민감도 슬라이더가 나타납니다. 감지 민감도는 과도 신호 시작 지점 감지 알고리즘의 민감도를 제어합니다.
  • Manual Markers를 선택하면 마커를 수동으로 배치합니다.

참고: 마커를 이동, 제거하거나 새로운 마커를 추가하면 모드가 수동으로 자동 변경됩니다.

Default value: Import From File
