Pan Left-Right (3D)

현재 선택된 조종점의 X축 상의 좌표를 말합니다. X 좌표는 감소 최대 거리의 백분율로써 측정되며, 그렇기 때문에 이 값은 (심지어 0이 될 수도 있는) 실재 거리값에 상관 없이 0과 +/- 100 사이로 정해집니다.

두 개 이상의 조종점이 선택된 경우 해당 영역은 0의 값으로 나타납니다. 이러한 방식으로 선택된 모든 조종점의 값을 그들의 현재값에 관해 늘이거나 줄일 수 있습니다. 예를 들어 두 개의 조종점을 선택한 후 X 글상자에 -5를 입력할 경우, 두 개의 조종점 모두 5퍼센트 단위로 왼쪽으로 옮겨집니다.

Default value: 0
Range: -100 to 100
