Track Type

현재 트랙과 그의 하위 트랙들의 재생 작동 방식을 정의합니다. 다음은 선택 가능한 Track Type입니다:

-Normal: 매번 상위 세그먼트가 재생될 때마다 현재 트랙을 재생합니다. 하위 트랙들은 Normal 모드가 선택된 경우 추가될 수 없습니다. -Random Step: 매번 상위 세그먼트가 재생될 때마다 하위 트랙 중 하나를 랜덤화된 순서로 재생합니다. -Sequence Step: 매번 상위 세그먼트가 재생될 때마다 하위 트랙 중 하나를 순차적으로 재생합니다. - Switch: 활성화된 Switch/State에 연관된 하위 트랙들을 재생합니다.

Default value: Normal
