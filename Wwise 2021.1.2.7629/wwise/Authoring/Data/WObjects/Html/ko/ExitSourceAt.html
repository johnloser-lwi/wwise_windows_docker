해당 소스의 어느 시점에 퇴장이 개시될지를 나타냅니다. 다음의 옵션 중에서 선택하실 수 있습니다:

  • Immediate: Source가 재생을 즉시 멈춥니다.
  • Next Grid: Source가 다음 격자선 간격에 재생을 중지합니다. 격자는 음악 오브젝트가 가상적으로 분할되는 임의의 방법입니다.
  • Next Bar: 해당 소스의 재생을 다음 마디에 중단합니다.
  • Next Beat: 해당 소스의 재생을 다음 박자에 중단합니다.
  • Next Cue: 사용자 정의 큐이든 퇴장 큐이든 상관없이 해당 소스의 재생을 다음 큐에 중단합니다.
  • Next Custom Cue: Source가 다음 맞춤 큐에 재생을 멈춥니다. 만약 현재 세그먼트에 사용자 정의 큐가 없을 경우 Wwise는 계속해서 사용자 정의 큐를 찾을 때까지 다음 세그먼트로 진행합니다.
  • Exit Cue: 해당 소스의 재생을 퇴장 큐에 중단합니다.

참고: ‘Next Cue’ 또는 ‘Next Custom Cue’를 선택하실 때에, 어떤 사용자 정의 큐의 이름을 부합시킬지를 정의하실 수 있습니다. 이때 이름은 정확히 일치해야 합니다. 명시된 이름이 없을 경우 어떤 사용자 정의 큐이든 부합시킬 수 있습니다.

이 옵션은 Music Switch Container들을 위해서만 사용 가능합니다.

Default value: Exit Cue
