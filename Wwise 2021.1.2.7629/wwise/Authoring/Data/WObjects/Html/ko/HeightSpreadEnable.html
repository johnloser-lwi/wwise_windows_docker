Enable Height Spread

Height Spread는 패닝이 평면 구성에서 발생할 경우에만 고도를 기준으로 최소 확산값을 계산하고, 7.1.4처럼 반구형 구성에서 패닝이 발생할 경우 음수 고도만을 기준으로 최소 확산값을 계산합니다. 이 설정은 3D 사운드가 리스너 위나 아래로 지나갈 때 패닝을 매끄럽게 만들어줍니다.

Height Spread는 사운드에 감쇠가 없을 경우에도 기본적으로 항상 활성화되어 있습니다. Height Spread는 이 속성을 선택 해제해야만 비활성화할 수 있습니다. 사운드에 감쇠가 없으며 Height Spread를 비활성화하려는 경우, 해당 목적을 위해 일정한 Output Bus Volume이 있는 Attenuation ShareSet를 반드시 생성해야 합니다.

더 자세한 정보는 Height Spread의 효과를 참고해 주세요.

Default value: true
