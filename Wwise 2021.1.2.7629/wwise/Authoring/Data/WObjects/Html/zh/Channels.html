Channels

声道数。您希望包含在转码文件中的音频声道数。

以下选项可用：

  • As Input – 同输入。将音频声道数保持与原始媒体文件相同。
  • Mono – 单声道。所有声道将混合为一个声道，但 LFE 除外，该声道将被弃用。
  • Mono drop – 单声道弃用。除了第一个声道外，其它所有声道都将被弃用。
  • Stereo – 立体声。所有声道都将混合到左前和右前声道，但 LFE 除外，该声道将被弃用。
  • Stereo drop – 立体声弃用。除了左声道和右声道外，其它所有声道都被弃用。

有关 Wwise 如何处理音频声道的下混转换的详细信息，请参阅 创建音频转码设置 ShareSet 。

Default value: As Input
