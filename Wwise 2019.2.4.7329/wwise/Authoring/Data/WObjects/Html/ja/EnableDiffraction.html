Enable diffraction

Spatial Audioによるディフラクション（回折）を、このサウンドで有効にする。

ディフラクションは、仮想環境にある音響的なポータルや、レベル内のジオメトリなど、障害物の周りを音が曲がるという音響現象のシミュレーションである。音のディフラクションには、レベルのジオメトリや、ルームやポータルが、ゲームで定義されていて、それがWwise Spatial Audioに送信される必要がある。

ある音にディフラクションが適用されると、Wwise Spatial Audioで以下が起きる:

  • エミッタからリスナーまでのパスを、1つ以上計算する。
  • ディフラクションの係数（0%-100%）を、パス内の角度の合計に基づいて計算する。パスが直線であればディフラクションを0%とみなし、パスが180°以上曲がればディフラクションを100%とみなす。
  • ディフラクションの係数（%）を、WwiseのProject Settingsで定義するobstructionカーブに基づいて、減衰値、ローパスフィルタ値、ハイパスフィルタ値に変換する。対応する減衰とフィルターを、サウンドに適用する。
  • 音とリスナーの見かけの位置関係を計算し、仮想ポジションを作成し、リスナーにとって音が角の向こう側から、またはポータル経由で、伝搬しているような印象を与える。

注釈:ユーザーは、3D Game Object ViewerでWwise Spatial Audio内の動きを見ることができ、例えばレベルのジオメトリ、ポータル、音のディフラクションパス、音の結果的な仮想ポジションなどが見える。

注釈:ディフラクションを有効にするには、Wwise Spatial Audioライブラリを初期化し、ゲームがWwise Spatial Audioにレベルのジオメトリ、またはルームやポータルを送信する必要がある。

Default value: false
