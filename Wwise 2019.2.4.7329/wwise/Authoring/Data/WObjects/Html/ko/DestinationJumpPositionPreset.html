음악 오브젝트의 어느 시점에 목적지가 재생을 시작할지를 정합니다. 다음의 옵션 중에서 선택하실 수 있습니다:

  • Entry cue: 목적지가 오브젝트의 도입부 큐부터 재생을 시작합니다.
  • Same time as playing segment: 목적지가 소스 세그먼트와 동일한 시점에서부터 재생을 시작합니다. 예를 들어 소스 세그먼트가 처음부터 15초를 재생했을 경우 목적지는 15초가 지난 지점부터 재생을 시작합니다.
  • Random Cue: 목적지가 랜덤으로 선택된 큐에서부터 재생을 시작합니다. 이 옵션이 선택된 경우 도입부 큐 또는 필터링 기준에 부합하는 모든 사용자 정의 큐를 포함한 어떤 큐이든지 선택될 수 있습니다. Match source cue name과 Match (jump to destination)를 참조해 주세요.
  • Random Custom Cue: 목적지가 필터링 기준에 부합하는 사용자 정의 큐 중에서 랜덤으로 선택한 큐에서부터 재생을 시작합니다. Match source cue name과 Match (specify cue name)을 참조해 주세요. 해당 세그먼트에 사용자 정의 큐가 없을 경우 도입부 큐가 사용됩니다.
  • Last Exit Position: 도착지가 마지막으로 재생한 세그먼트의 경과 시간으로 상쇄하여 재생을 시작합니다. 예를 들어 마지막으로 도착지의 세 번째 세그먼트의 15초까지 재생을 했었다면 도착지가 15초부터 재생을 시작합니다.

참고: Last Exit Position 옵션은 동일한 재생 인스턴스 동안 상위 Music Container의 전환 효과에서만 적용됩니다.

Default value: Entry Cue
