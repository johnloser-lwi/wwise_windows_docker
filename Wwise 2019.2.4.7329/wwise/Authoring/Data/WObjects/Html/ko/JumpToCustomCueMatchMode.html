Match source cue name

어떤 사용자 정의 큐가 목적 세그먼트 내에서 시작 지점으로써 쓰일 수 있는지를 정합니다. 과도 구간의 소스에 사용된 큐의 이름과 동일한 이름을 가진 사용자 정의 큐만 시작 지점 후보로 여겨집니다. 예를 들어 해당 소스가 ‘A’라는 이름의 사용자 정의 큐에서 퇴장할 경우, 목적지는 ‘A’라고 이름지어진 사용자 정의 큐에서부터 재생을 시작하게 됩니다.

문자열 비교는 대/소문자 구분이 없지만 문자는 정확히 일치해야 합니다.(와일드 카드 없음).

이 옵션은 ‘Sync to’가 ‘Random Cue’ 또는 ‘Random Custom Cue’로 설정되어 있을 때에만 사용하실 수 있습니다.

부합되는 큐를 찾지 못할 경우 도입부 큐가 사용됩니다.

이 옵션은 현재 다음의 상황에만 작동합니다T:

  • ‘Exit Source At’ 설정이 ‘Next Cue’ 또는 ‘Next Custom Cue’로 설정되어 있을 때.
  • ‘Source’ 오브젝트가 ‘nothing’이 아닐 경우.

이름 일치는 사용자 정의 큐의 선택에만 영향을 끼치며 도입부 큐의 자격 요건에는 전혀 영향을 끼치지 않습니다. ‘Random Cue’를 ‘Sync to’ 속성으로써 사용하실 경우, 선택된 목적지 큐는 소스가 퇴장한 큐와 동일한 이름을 가진 큐 또는 도입부 큐로 선택됩니다. 도입부 큐를 가능한 목적지 후보에서 제외하기 위해서는 ‘Random Custom Cue’를 ‘Sync to’ 속성으로 사용하시면 됩니다.

Match (specify cue name) 어떤 사용자 정의 큐가 목적 세그먼트 내에서 시작 지점으로써 쓰일 수 있는지를 정합니다. 사용자 정의 큐들 중에서 편집 상자 내 문자열을 부합하는 큐들만 시작 지점 후보로 여겨집니다. 명시된 이름이 없을 경우 어떤 사용자 정의 큐이든 부합시킬 수 있습니다.

문자열 비교는 대/소문자 구분이 없지만 문자는 정확히 일치해야 합니다.(와일드 카드 없음).

이 옵션은 ‘Sync to’가 ‘Random Cue’ 또는 ‘Random Custom Cue’로 설정되어 있을 때에만 사용하실 수 있습니다.

부합되는 큐를 찾지 못할 경우 도입부 큐가 사용됩니다.

이름 일치는 사용자 정의 큐의 선택에만 영향을 끼치며 도입부 큐의 자격 요건에는 전혀 영향을 끼치지 않습니다. ‘Random Cue’를 ‘Sync to’ 속성으로 사용하실 경우, 선택된 목적지 큐는 ‘Match’ 텍스트 란에 입력된 문자열과 상응하는 이름의 사용자 정의 큐 또는 도입부 큐로 선택됩니다. 도입부 큐를 가능한 목적지 후보에서 제외하기 위해서는 ‘Random Custom Cue’를 ‘Sync to’ 속성으로 사용하시면 됩니다.

Default value: Match specific name
