Input

이 매개 변수는 어떤 채널이 피치 시프터에 의해 처리될지를 정합니다.

Values:

  • As Input: 모든 입력 채널을 처리합니다 (기본 설정).
  • Mono/Center: 중앙 채널만 처리합니다. 모노 신호 또한 처리합니다.
  • Stereo: 좌측과 우측 스피커만 처리합니다.
  • L/R/C: 중앙 스피커가 있을 경우 중앙 스피커까지 모든 앞쪽 스피커를 처리합니다.
  • L/R/Ls/Rs: 앞쪽과 뒷쪽 스피커 쌍을 처리합니다.
  • L/R/C/Ls/Rs: 모든 전채 대역 채널을 처리합니다.

Default value: As Input
