Filter Type

각 그레인에 적용되는 필터의 유형을 말합니다.

  • LPF6: 1차 (6 dB/옥타브) 저역 통과 필터(low-pass filter). 이 필터를 사용하면 매개 변수 Filter Q가 아무 효과를 주지 않습니다.
  • LPF12: 2차 (12 dB/옥타브) 저역 통과 필터(low-pass filter).
  • HPF6: 1차 (6 dB/옥타브) 고역 통과 필터(high-pass filter). 이 필터를 사용하면 매개 변수 Filter Q가 아무 효과를 주지 않습니다.
  • HPF12: 2차 (12 dB/옥타브) 고역 통과 필터(high-pass filter).
  • BP: 2차 (6 dB/양쪽 옥타브) 대역 통과 필터(band-pass filter). 이 필터를 사용하면 매개 변수 Filter Q가 아무 효과를 주지 않습니다.

Default value: LPF12
