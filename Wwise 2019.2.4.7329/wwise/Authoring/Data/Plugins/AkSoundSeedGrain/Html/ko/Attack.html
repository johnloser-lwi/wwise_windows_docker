Attack

그레인 엔벨로프의 어택 시간을 밀리초(ms)로 표시합니다. Window Mode가 Release Same as Attack일 경우 Release 시간이 Attack 시간과 같아집니다.

Attack 엔벨로프는 그레인의 시작 지점에서 시작하기 때문에 그레인의 재생 시간에 포함됩니다. 반면에 엔벨로프의 릴리스 부분(Release)은 그레인의 끝 부분에서 시작하기 때문에 그레인의 재생 시간에 포함되지 않습니다.

또한 그레인 비주얼라이저 창의 왼쪽 위에 있는 손잡이를 사용하면 Attack 시간을 변경할 수 있습니다.

참고: Attack 시간이 그레인의 재생 시간보다 길 경우 그레인은 전체 볼륨이 아닌 Attack이 절단되었을 때 적용된 엔벨로프의 레벨로 재생됩니다.

Default value: 10
Range: 0 to 5000
