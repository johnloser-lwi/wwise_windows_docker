Duration Multiplier

그레인의 재생 시간을 방사율(Emissions per Second 혹은 Time Between Emissions를 사용하여 설정)의 배율로 표현합니다. 이 컨트롤은 Duration Link가 Duration Multiplier로 설정되었을 경우에만 사용할 수 있습니다.

값이 1로 설정될 경우 실제 그레인의 재생 시간은 각 그레인 사이에 경과하는 시간과 동일합니다. 그래서 엔벨로프의 릴리스가 엔벨로프 어택과 겹치면서 그레인이 계속해서 서로 맞물립니다. 값이 1보다 작을 경우 그레인 사이에 빈 공간이 생기며, 값이 1보다 클 경우 그레인이 계속 쌓이게 됩니다.

Default value: 1
Range: 0.001 to 1000
