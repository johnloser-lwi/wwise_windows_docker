Duration Multiplier Quantization

Duration Multiplier에 적용된 변조의 퀀티제이션을 말합니다.

이 속성에 사용할 수 있는 퀀티제이션의 종류는 다음과 같습니다.

  • None: 퀀티제이션을 하지 않음.
  • (-1, 1): 변조기(modulator) 파형의 위와 아래로 퀀티제이션함.
  • (-1, 0, 1): 변조기 파형의 위, 중간, 아래로 퀀티제이션함. ‘중간’은 양극성 파형에만 적용되며 기반 Duration Multiplier 값에 해당합니다.

Default value: None
