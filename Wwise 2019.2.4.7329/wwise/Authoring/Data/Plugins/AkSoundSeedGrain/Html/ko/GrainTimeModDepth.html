Time Between Emissions Modulation

Time Between Emissions에 적용되는 변조량을 말합니다.

변조량은 변조기(modulator)에서 선택된 파형에 따라, 원래 값보다 높거나 낮음을 옥타브의 개수로 표현합니다. 옥타브를 한 개 높이면 재생 시간이 절반이 되며, 옥타브를 한 개 낮추면 재생 시간이 두 배가 됨을 의미합니다. 음수 값은 변조기의 출력 신호를 뒤집는 효과를 냅니다 (예: 극성을 반전시킴).

참고: Time Between Emissions Mod 1의 값은 Emissions Per Second Mod 1의 값과 동일합니다. 다시 말해 Select Freq/Time이 Frequency였을 경우 동일한 값으로 동일한 변조가 일어납니다.

Default value: 0
Range: -10 to 10
