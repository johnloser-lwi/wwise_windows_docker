Select Freq/Time

그레인 방사 빈도/주파수(Emissions per Second)를 제어할지, 각 그레인이 방사되는 사이의 시간(Time Between Emissions)을 제어할지, 혹은 MIDI 노트로 그레인 방사 빈도/주파수(MIDI Emissions per Second)를 제어할지를 선택합니다.

MIDI Emissions Per Second를 선택했으며 신시사이저가 MIDI 노트에서 생성된 경우 노트의 값이 Emissions per Second 값을 덮어 씁니다. Emissions per Second 속성을 변조하면 이 값이 변조됩니다.

Default value: Time between Emissions
