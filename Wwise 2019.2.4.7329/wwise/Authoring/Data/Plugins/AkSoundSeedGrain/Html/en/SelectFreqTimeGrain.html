Select Freq/Time

Select whether you want to control the grain emission frequency (Emissions per Second), the time between each grain is emitted (Time Between Emissions) or to control the grain emission frequency from a MIDI note (MIDI Emissions per Second).

When MIDI Emissions Per Second is chosen, and if the synthesizer was spawned from a MIDI note, the note value overrides Emissions per Second. Modulating the Emissions per Second property will modulate this value.

Default value: Time between Emissions
