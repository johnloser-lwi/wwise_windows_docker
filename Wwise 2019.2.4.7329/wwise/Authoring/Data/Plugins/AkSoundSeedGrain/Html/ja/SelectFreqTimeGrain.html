Select Freq/Time

制御したいのが、グレイン発生の頻度（Emissions per Second）なのか、グレイン同士の発生の間隔なのか（Time Between Emissions）、それともMIDIノートからグレイン発生頻度を制御することなのか（MIDI Emissions per Second）を選択。

MIDI Emissions Per Secondを選択し、シンセサイザーがMIDIノートからスポーンされた場合は、ノート値が、Emission per Secondより優先される。Emissions per Secondプロパティをモジュレーションすると、この値がモジュレーションされる。

Default value: Time between Emissions
