Output Spread

对于标准声道配置（立体声、5.1 等），Output Spread 将决定发送至左侧声道（左前和左后）和右侧声道（右前和右后）混响信号的相似性。这样就可以通过扩大/缩小信号混响（湿声部分）的散布范围来增强/减弱空间感。

当使用零值时，左声道和右声道的混响输出相同。 在值为 180 时，则会将多声道卷积运算的结果直接馈送至各独立声道。

对于 Ambisonics 配置，扩散并不只是横向的（立体声），而是遍布所有方向。实际上，它会改变各方向声道与全向声道 W 之间的比重。

备注:当通过耳机监视时该效果会更明显，因为耳机没有会减弱效果的扬声器串扰（cross-talk）。

警告:该属性对于 1.0（单声道）或 1.1 声道声音没有效果。

警告:此属性不适用于 Filter 模式。

单位：°

Default value: 180
Range: 0 to 180
