IR Threshold

阈值。指定一个电平，低于该电平时，冲激响应中所含的能量将被丢弃，以节省内存和 CPU 资源。 在计算特定频率的能量时会与该阈值相比较，由于特定频率的能量在冲激响应中随着时间而变化，这样做可优化卷积的存储和处理，而将品质损失降至最低。 值为 -144 dB 将处理全部冲激响应信息，而更低的值通常会消除能量低的高频信息，以获得显著优化。 阈值越高（如超过 -50 dB），压缩导致的副作用越明显。 建议您在不显著影响音频品质的情况下，将阈值设得尽可能高。 要了解详情，请参阅 内存和 CPU 优化 。

Default value: -144
Range: -144 to -30
