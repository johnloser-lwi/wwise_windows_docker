Reverb Type

사용하는 알고리즘의 유형을 결정합니다.

Reverb (기본) 이 모드에서는 다채널의 내용물이 Input Levels의 값을 이용하여 먼저 다운믹스됩니다. 다채널 중첩이 실행된 후 이 결과가 Output Levels 컨트롤을 사용해 원본 다채널 구성으로 다시 업믹스됩니다. 확산, 초기 지연, 뒷 채널 지연과 같은 추가적인 처리또한 최적량의 메모리와 CPU를 사용하여 수행됩니다.

작은 정보: 어떤 목적을 위한 잔향이든 모두 이 작동 모드를 사용해야 합니다.

Filter: 이 모드를 사용하면 각 채널과 임펄스 응답 채널(들) 간에 직접 컨볼루션을 적용할 수 있습니다. 이 모드는 잔향 모드와 달리 다운믹스 또는 업믹스가 실행되지 않기 때문에 Input/Output Levels가 비활성화되어있습니다. 표준 다채널 임펄스 응답의 경우 좌측 입력 채널 (앞, 옆 및/또는 뒤)은 좌측 임펄스 응답으로 중첩되며 우측 입력 채널은 우측 임펄스 응답 채널로 중첩됩니다.

작은 정보: 이 모드를 사용하면 복잡한 주파수 응답 (예를 들어 전자 기기 또는 벽돌 필터링을 수행하기 위한 보정된 임펄스 응답)을 모든 입력 채널에 적용할 수 있습니다.

Default value: Reverb
