Output Spread

표준 채널 구성의 경우 (스테레오, 5.1 등), Output Spread는 좌측 채널 (앞 및/또는 뒤)과 우측 채널 (앞 및/또는 뒤)에 전송되는 잔향의 유사성을 결정합니다. 이는 신호의 잔향 (변조 신호)의 확산을 확대/축소함으로써 공간 감각을 높이거나 줄여줍니다.

이 값이 0일 때에 좌측과 우측 채널의 잔향 출력은 동일합니다. 값이 180일 때에 다채널 중첩의 결과가 직접 각 방향으로 공급됩니다.

앰비소닉의 경우 확산은 측면으로만 (스테레오) 일어나지 않고 모든 방향으로 일반화됩니다. 사실상 이는 옴니 채널 W와 비교해 지향성 채널에 더 가중치를 둡니다.

참고: 이 효과는 헤드폰을 통해 모니터링할 경우 더 강력하게 나타나는데, 왜냐하면 스피커 신호 사이 혼선의 부재가 이 Effect를 감소하기 때문입니다.

주의: 이 속성은 1.0 (모노) or 1.1 채널 사운드에는 영향을 끼치지 않습니다.

주의: 이 속성은 Filter 모드에는 적용되지 않습니다.

단위: °

Default value: 180
Range: 0 to 180
