Input Spread

잔향 처리 전에 좌측과 우측 다운믹스 간의 혼선의 양을 결정합니다. 이는 연관성이 강한 임펄스 응답의 입력 다운믹스의 확산을 증가함으로써 공간 감각을 넓힐 수 있습니다.

값이 0일 때에 입력 신호는 모노로 다운 믹스된 후 각 임펄스 응답 채널로 중첩됩니다. 값이 180일 때에 다채널 중첩의 결과가 직접 각 방향으로 공급됩니다.

참고: 반면 입력 확산이 180 (최대)일 경우 완전히 좌측으로 패닝된 수신 신호가 임펄스 응답의 좌측 채널로 완전히 중첩되는데, 이럴 경우 소리의 전달에 있어 비현실적입니다.

작은 정보: 다채널 임펄스 응답의 경우 Reverb 모드에서 모노 또는 유사 모노(quasi-mono) 다운믹스 (Input Spread가 0과 가까운)를 사용하는 것을 권장합니다. 높은 값의 Input Spread는 모노 임펄스 응답의 경우에만 사용해야 하는데, 왜냐하면 이는 왼쪽과 오른쪽에 다른 신호 (공간 감각)을 창조할 수 있는 단 한 가지 방법이기 때문입니다.

주의: 이 속성은 1.0 (모노) or 1.1 채널 사운드에는 영향을 끼치지 않습니다.

주의: 이 속성은 Filter 모드에 적용되지 않습니다.

주의: 이 속성은 앰비소닉 임펄스 응답 구성의 경우 무시됩니다.

참고: Input Spread는 앰비소닉 입력 신호와 모노 임펄스 응답과 함께 사용할 수 있습니다.

Default value: 0
Range: 0 to 180
