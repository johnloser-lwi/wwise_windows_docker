IR Channels

다음은 다채널 임펄스 응답에 사용 가능한 다운믹스 옵션입니다.

  • As Input: 임펄스 응답의 모든 채널을 사용하여 다채널 컨볼루션을 실행합니다.
  • Stereo: 임펄스 응답의 스테레오 다운믹스를 사용하여 스테레오 컨볼루션을 실행합니다. 무명 채널 구성은 사용할 수 없습니다 (As Input으로 다시 전환됨).
  • Mixdown: 임펄스 응답 내 모든 채널의 다운 믹스를 사용하여 모든 채널의 모노 컨볼루션을 실행합니다. 무명 채널과 앰비소닉 채널의 경우 첫 번째 채널 / W를 사용합니다.
  • Left only: 임펄스 응답의 왼쪽 채널을 사용하여 모든 채널의 모노 컨볼루션을 실행합니다. 무명 채널과 앰비소닉 채널의 경우 첫 번째 채널 / W를 사용합니다.
  • Right only: 임펄스 응답의 오른쪽 채널을 사용하여 모든 채널의 모노 컨볼루션을 실행합니다. 무명 채널과 앰비소닉 채널의 경우 첫 번째 채널 / W를 사용합니다.

참고: 이 컨트롤은 다채널 임펄스 응답 파일을 사용할 때에만 사용할 수 있습니다.

Default value: As Input
