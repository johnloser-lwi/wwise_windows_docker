マルチチャンネルインパルス応答用に利用可能なダウンミックスオプション:

  • As Input:複数チャンネルコンボリューションを実行するために、すべてのインパルス応答チャンネルを使う。
  • Stereo:ステレオコンボリューションを実行するため、インパルス応答のステレオダウンミックスを使う。anonymousチャネル構成では無視される。（As Input に戻る）
  • Mixdown:インパルス応答の総チャンネルのダウンミックスを使用して、全てのチャンネルでモノコンボリューションを実行。Anonymous 構成とambisonicsが最初のチャンネル / W を取得する。
  • Left only:インパルス応答の左チャンネルを使用して、全てのチャンネルでモノコンボリューションを実行。Anonymous 構成とambisonicsが最初のチャンネル / W を取得する。
  • Right only: インパルス応答の右チャンネルを使用して、全てのチャンネルでモノコンボリューションを実行。Anonymous 構成とambisonicsが最初のチャンネル / W を取得する。

注釈:この制御は、マルチチャンネルインパルス応答ファイルが使用されいてる場合にのみ有効になる。

Default value: As Input
