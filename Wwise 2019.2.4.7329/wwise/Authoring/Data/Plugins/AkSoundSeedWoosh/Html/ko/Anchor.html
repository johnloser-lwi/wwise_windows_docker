이동하는 오브젝트가 주어진 전향기 주변으로 회전하거나 도는지를 결정합니다. 고정점을 명시하면 전향 샘플이 다양한 풍속의 바람을 직면할 수 있도록 해줍니다. 고정점을 정의하지 않을 경우 모든 전향 샘플은 동일한 속도로 움직인다고 가정됩니다.

예를 들어 세 개의 전향기를 가진 오브젝트가 있는데 이중 첫 번째 전향기가 고정점으로 정의되어 있다고 해봅시다.

  • Deflector 1: 속도가 없습니다 (움직이지 않습니다).
  • Deflector 2: 편향기 3의 절반의 속도로 이동합니다.
  • Deflector 3: Object 속도 속성값이 정의한 전속력으로 이동합니다.

Default value: false
