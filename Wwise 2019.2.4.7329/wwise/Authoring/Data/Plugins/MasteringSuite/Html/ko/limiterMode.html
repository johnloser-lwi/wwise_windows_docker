Limiter Mode

  • Hard-Knee Limiter: Audio signals will be cut off at the threshold value.
  • Soft-Knee Limiter: Temporarily allow the threshold to be exceeded, and gradually change the audio signal back to the threshold level over the specified attack time.
  • Advanced Limiter: Automatically configure the attack and react to peaks smoothly before the input reaches the threshold, with introduction of a 2.67ms latency. The signal is kept under the threshold.

Default value: Soft
