Threshold Mode

Continous (지속형) 모드는 움직임으로 인해 일어나는 최대 허용 Doppler(도플러) 피치 시프트를 허용합니다. 움직임이 한계점을 초과할 만큼 빠를 경우 반사의 볼륨은 음조가 다시 한계점 아래로 돌아올 때까지 덕킹됩니다. 한계점이 0일 경우 움직임이 일어날 때마다 반사의 볼륨이 덕킹됩니다.

Step(단계형) 모드는 최소 이동량이 일어날 때까지 반사음을 업데이트하지 않는 거리 한계점 ‘버블’을 허용합니다. 한계점이 0일 경우 움직임이 일어날 때마다 이전의 반사와 새로운 반사가 서로 크로스페이드하게 됩니다. 크로스페이드는 음조를 변경하지 않는 지속 시간 변경 효과를 일으키게 됩니다.

Default value: Continuous
