Stereo Width

立体声宽度。决定在左声道（前或后）和右声道（前或后）中输出的 ER + 混响内容之间的相似性。 这可以形成更强的空间感，方法是扩大早期反射和混响的立体声散布（Spread）。

当使用零值时，左声道和右声道的 ER + 混响输出相同。 因此可获得一定的内存和 CPU 优化。 值为 180 表示各侧收到完全不同的 ER + 混响信号。 当通过耳机监视时该效果会更明显，因为耳机没有会减弱效果的扬声器串扰（cross-talk）。

该属性对于 1.0（单声道）或 1.1 声道声音没有效果。

单位：°

Default value: 180
Range: 0 to 180
