Time

곡선 상 각 지점의 시간 속성값에 적용되는 절대 상쇄값을 말합니다. 이 값은 각 지점의 가능한 시간값의 범위를 정의합니다. 예를 들어 속성값에 1의 랜덤값을 추가할 경우 각 조종점의 가능한 시간값은 [(값 - 1)부터 (값 +1)]이 됩니다.

Random 값을 정의할 때, Wwise는 ‘쉭’ 소리가 매 회 재생될 때 마다 이 범위 (값 +/- 랜덤)에서 무작위로 값을 선택합니다. 이는 곡선 상 각 지점을 왼/오른쪽으로 각각 움직일 수 있도록 합니다.

이 조종점의 순서는 변경될 수 없는데 이는 곡선의 윤곽을 유지하기 위함입니다. 그렇기 때문에 각 점들은 바로 전 지점과 바로 다음 지점 사이로만 랜덤으로 이동할 수 있습니다.

Default value: .5
Range: 0 to 1
