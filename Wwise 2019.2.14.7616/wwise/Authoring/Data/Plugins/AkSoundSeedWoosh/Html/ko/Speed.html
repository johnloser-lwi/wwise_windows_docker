Object Speed

가장 빠르게 이동하는 편향기가 공기 중에서 이동하는 평균 속도를 말합니다. 고정점이 있을 경우, 고정점에서 가장 멀리 떨어진 전향기가 가장 빨리 이동합니다.

Object speed는 씬의 전향기에 적용되는 에너지의 상쇄값으로 해석됩니다. 이 속성은 근본적으로 전향기의 공진 주파수의 속도 효과를 정의합니다.

기본적으로 Object Speed 속성은 Automate 옵션이 선택되어 있습니다. 자동화 곡선은 (t=0, 0), (t=0.25, 100), (t=0.5, 0) 이 세가지의 점들을 포함합니다.

Default value: 0
Range: -50.0 to 50.0
