Type

적용될 디스토션 알고리즘의 유형을 나타냅니다. 다음은 사용 가능한 옵션입니다:

  • None: 디스토션이 적용되지 않습니다.
  • Overdrive: 부드러운 블루스 유형의 디스토션을 적용합니다.
  • Heavy: 다량의 고주파 배음을 가진 억센 디스토션을 적용합니다.
  • Clip: 전이가 없는 억센 클리핑을 적용합니다. 때때로 원치 않는 잡음을 일으킬 수 있습니다 (클리핑 디스토션).
  • Fuzz: 양수값과 음수값에 다른 한계치를 가진 억세면서 부드러운 비대칭적 클리핑을 적용합니다. 이 유형의 디스토션은 많은 배음을 가지며 복잡한 음이 들어올 때 빠른 속도로 불협화가 생길 수 있습니다.

Default value: None
