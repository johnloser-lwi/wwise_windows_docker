변환된 소스에 포함하고자 하는 오디오 채널의 갯수를 나타냅니다.

다음은 사용 가능한 옵션입니다:

  • As Input - 미디어 파일의 원본과 같은 갯수의 오디오 채널을 유지합니다.
  • Mono - 모든 채널이 하나의 채널로 합쳐집니다, 단 LFE(Low Frequency Effect, 저주파 효과) 채널은 제외됩니다.
  • Mono drop - 첫 번째 채널을 제외한 모든 채널을 제외합니다.
  • Stereo - 모든 채널이 왼쪽 앞과 오른쪽 앞 채널로 합쳐집니다, 단 LFE 채널은 제외됩니다.
  • Stereo drop - Left와 Right로 정의된 채널을 제외한 모든 채널이 제외됩니다.

와이즈의 오디오 채널 변환 다루기에 대한 더 자세한 내용은 오디오 Conversion Settings ShareSet 만들기 를 참조해 주세요.

Default value: As Input
