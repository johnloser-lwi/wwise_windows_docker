First Occurrence Only

해당 Switch Container가 게임 엔진으로부터 촉발되었을 때마다 오브젝트를 재생할 것인지 아니면 Switch가 바꼈을 때에만 재생할 것인지를 정합니다.

First Occurrence Only 옵션이 활성화된 경우 해당 오브젝트는 Switch가 처음 변경될 때에만 재생됩니다. 같은 Switch Container의 Subsequent Trigger는 오브젝트를 재생하지 않을 것입니다. First Occurrence Only 옵션이 활성화된 경우 해당 오브젝트는 게임 엔진이 Switch Container을 트리거할 때마다 재생됩니다.

Default value: false
