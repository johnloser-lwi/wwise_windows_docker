Scope

어떻게 LFO 인스턴스가 생성되는가를 정의합니다:

  • Voice(보이스): LFO의 인스턴스가 모든 사운드/오브젝트의 재생을 위해 생성됩니다.
  • Note/Event(노트/이벤트): 미디의 맥락에서 사용되었을 때에, LFO의 인스턴스가 재생 중인 모든 인스턴스 또는 노트를 위해 생성됩니다.
  • Game Object(게임 오브젝트): LFO의 인스턴스가 각 게임 오브젝트를 위해 생성됩니다.
  • Global: 전체 프로젝트를 위해 하나의 LFO가 생성됩니다.

Default value: Note or Event
