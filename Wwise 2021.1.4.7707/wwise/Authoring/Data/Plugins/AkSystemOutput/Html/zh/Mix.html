Mix Behavior

混音行为。提供以下选项：

  • Use Default：允许 Wwise 选择最适合此 Audio Object 的目标。有关更多详细信息，请参阅了解基于对象的音频。
  • Mix to Main：强制将 Audio Object 发送到 Audio Device 的 Main Mix。
  • Mix to Passthrough：强制将 Audio Object 发送到 Audio Device 的 Passthrough Mix。若 Audio Device 不支持 Passthrough，则将 Audio Object 发送到 Main Mix。

Default value: Use Default
