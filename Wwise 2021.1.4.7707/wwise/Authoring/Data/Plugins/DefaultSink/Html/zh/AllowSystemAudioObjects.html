Allow System Audio Objects

允许系统音频对象。在激活 3D Audio 并启用该选项的情况下，若终端支持的 Audio Object 数量大于指定的阈值（即 Minimum System Audio Objects Required 的值），则允许 Audio Device 将 System Audio Object 发送到终端。

若禁用该选项，则禁止发送各个 System Audio Object。

在禁用时，会将 Audio Object 混音到 Main Mix，同时保留其空间特性。在这种情况下，最终的空间精度将取决于 Main Mix 的配置。

Default value: true
