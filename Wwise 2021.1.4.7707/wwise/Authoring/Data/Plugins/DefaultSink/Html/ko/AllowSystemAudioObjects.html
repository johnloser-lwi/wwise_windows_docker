Allow System Audio Objects

3D 오디오가 활성화되었을 때에 이 옵션을 선택할 경우 오디오 종단점이 지정된 한계점(예: Minimum System Audio Objects Required)보다 더 많은 Audio Object를 지원할 경우 Audio Device가 종단점으로 System Audio Object를 전송할 수 있게 해줍니다.

개별적인 System Audio Object를 전송하지 않으려면 이 옵션을 선택 해제하세요.

이 옵션을 선택하지 않으면 Audio Object가 공간적 특성을 유지한 채로 Main Mix로 믹싱됩니다. 결과적인 공간적 정확도는 Main Mix의 구성에 따라 다릅니다.

Default value: true
