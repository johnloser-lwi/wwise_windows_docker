Mode

様々な入力信号や用途に合った様々なリミティングスタイルを作成するために使用される二次検出のタイプ。次のようなオプションがある:

  • クリーン（Clean） - 最も透明度の高いリミッタモード。測定可能ディストーションを最少量に保ちながら信号レベルを調整する。
  • ソフト（Soft） - クリーンモードより少し大きめでありながら、非常に透明度が高いモード。
  • スマート（Smart） - 信号ディストーションを最小に保ちながら信号レベルをソフトモードより大きくできるインテリジェントなリミティング。
  • ダイナミック（Dynamic） - スマートモードより音量が大きく、特定の条件下でコンプレッションポンピングをわずかに追加するモード。
  • ラウド（Loud） - 信号ディストーションを最小に保ちながら、信号レベルを可能な限り大きくできるモード。
  • クラッシュ（Crush） - いくらかの信号ディストーションを伴いながらラウドモードより信号レベルを大きくできるモード。

Default value: Clean
